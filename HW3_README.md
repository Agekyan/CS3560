## Homework 3 Scope
HW3 focuses on design artifacts for WSS (Wilderness Survival System):
- Class design (classes, attributes, methods, relationships)
- State modeling for complex object behavior
- Use cases and interactions
- Sequence models for typical flows
- Activity-level process descriptions

## Deliverables Included
- Main design writeup: docs/HW3_Design.md
- Homework folder overview: hw3/README.md
- HW3 iterations and changes: hw3/ITERATIONS.md
- HW3 file-level explanation: hw3/FILES.md

## Design Summary
The HW3 package defines a WSS design centered around:
- Map and terrain costs
- Player resources and movement/rest rules
- Item and trader systems
- Vision and Brain strategy objects
- Inheritance/polymorphism across terrain, item, trader, vision, and brain hierarchies
