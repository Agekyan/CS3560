# HW3 Text Package

This repository now contains a C++ project scaffold for HW4 and a text-based design package for HW3.

- Project root: `CMakeLists.txt`
- Source entry point: `src/main.cpp`
- Public headers: `include/wss/`
- HW3 design notes: `docs/HW3_Design.md`

The HW3 file is written as a prose substitute for the required UML deliverables:
- class diagram text
- use case text
- sequence model text
- activity diagram text
- state diagram text

The next step for the class assignment would be to convert those sections into actual UML diagrams if your instructor expects images instead of prose.
